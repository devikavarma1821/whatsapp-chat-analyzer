# whatsapp-chat-analyzer
WhatsApp Chat Analyzer
WhatsApp Chat Analyzer is a Python-based application that allows users to gain deep insights into their WhatsApp chat history. By analyzing exported chat files (.txt format), the tool can generate various visualizations and reports to help understand communication patterns and trends.

Key Features
Chat Statistics: Total messages, words, media, and links shared in the chat.
User Activity Analysis: Identify the most active participants and their messaging patterns over time.
Word Cloud Generation: Visual representation of the most frequently used words in the chat.
Media Analysis: Count and categorize images, videos, audio, and documents shared.
Emoji Analysis: Highlight commonly used emojis to understand emotions and reactions.
Message Timeline: Visualize daily, weekly, and monthly chat activity trends.
Export Reports: Export insights and statistics in CSV or PDF formats for further analysis.
Streamlit Interface: Easy-to-use interactive web interface to visualize results dynamically.
Technologies Used
Python – Core programming language.
Streamlit – For building interactive web dashboards.
Matplotlib & Seaborn – For data visualization.
WordCloud – To generate word cloud graphics.
Pandas & NumPy – For data analysis and manipulation.
This project is useful for anyone who wants to analyze personal chats, study communication patterns, or generate visual summaries for research or personal use. It is also a good demonstration of data analysis, visualization, and Python project development skills.
